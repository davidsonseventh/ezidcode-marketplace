# Project Summary
The project is a modern web application built using React and Vite, designed to provide a user-friendly interface for managing various dashboard functionalities, including admin controls, POS systems, and vendor management. It aims to streamline operations for businesses by integrating essential tools in a single platform.

# Project Module Description
The application consists of several functional modules, including:
- **Dashboard**: Admin and vendor dashboards for managing operations.
- **POS System**: Point of Sale functionalities for transaction management.
- **PPOB System**: Payment Point Online Bank integration.
- **UI Components**: A collection of reusable UI components such as buttons, forms, and dialogs.

# Usage
To set up the project:
1. Clone the repository.
2. Navigate to the project directory.
3. Install dependencies:
   ```bash
   pnpm install
   ```
4. Build the project:
   ```bash
   pnpm run build
   ```
5. Serve the static files using a web server of your choice.
